# 編譯和執行
如果你使用Unix-like作業系統和g++的話，執行以下指令來編譯：
```bash
$ make
```

執行每個執行檔：
```bash
$ ./solution12
$ ./solution13
$ ./solution14
```

NOTE: 要終止輸入的話，請按下`Ctrl+d`(EOF)。
